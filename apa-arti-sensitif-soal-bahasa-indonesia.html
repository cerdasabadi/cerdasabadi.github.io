# Python

Aturan Penulisan Sintaks Python

Setelah mempersiapkan segala perlengkapan untuk coding python dan mengetahui cara membuat program python, selanjutnya mari kita pelajari tentang aturan-aturan penulisan sintaks Python yang harus dipatuhi.

“Bagaimana kalau tidak dipathui?”

Resikonya bisa terjadi error .

Pada artikel ini, saya akan membahas beberapa aturan dasar penulisan sintaks Python yang harus diketahui.

Agar nanti mudah dalam menulis program.

Apa saja aturan-aturannya?

Silahkan di simak…

1. Penulisan Statement Python

Statement adalah sebuah intruksi atau kalimat perintah yang akan dieksekusi oleh komputer.

Contoh:

print("Hello World!")

print("Belajar Python dari Nol")

nama = "petani kode"

Penulisan satu statement tidak diakhiri dengan tanda titik-koma.

Sedangkan, bila kita ingin menulis lebih dari satu statement dalam satu baris, maka kita harus memisahnya dengan titik-koma.

Contoh:

print("Hello"); print("World"); print("Tutorial Python untuk Pemula")

nama_depan = "petani"; nama_belakang = "kode"

Tapi…

Menurut beberapa style guide python, tidak dianjurkan menulis lebih dari satu statement dalam satu baris. Karena akan sulit dibaca.

2. Penulisan String pada Python

String adalah teks atau kumpulan dari karakter.

String dalam pemrograman biasanya ditulis dengan dibungkus menggunakan tanda petik.

Bisa menggunakan tanda petik tunggal maupun ganda.

Contoh:

judul = "Belajar Pemrograman Python sampai Bisa"

penulis = 'Petani Kode'

Atau kita juga bisa menggunakan triple tanda petik.

Contoh:

judul = """Belajar Python dengan Cepat"""

penulis = '''Petani Kode'''

3. Penuilsan Case pada Python

Sintak Python bersifat case sensitive, artinya teksini dengna TeksIni dibedakan.

Contoh:

judul = "Belajar Dasa-dasar Python"

Judul = "Belajar Membuat Program Python"

Antara variabel judul dengan Judul itu dibedakan…

Case Style

Menurut rekomendasi style guide Google, berikut ini contoh penulisan case yang disarankan:

## Snake Case digunakan pada:

module_name, package_name, method_name, function_name, , global_var_name, instance_var_name, function_parameter_name, local_var_name.

## CamelCase digunakan Pada:

ClassName, ExceptionName

## ALL CAPS digunakan Pada:

GLOBAL_CONSTANT_NAME

Baca juga: 4 Macam Gaya Penulisan Case dalam Pemrograman

4. Penulisan Blok Program Python

Blok program adalah kumpulan dari beberpaa statement yang digabungkan dalam satu blok.

Penulisan blok program harus ditambahkan indentasi (tab atau spasi 2x/4x).

Blok Program

✔️ Contoh yang benar:

# blok percabangan if

if username == 'petanikode':

print("Selamat Datang Admin")

print("Silahkan ambil tempat duduk")

# blok percabangan for

for i in range(10):

print i

❌ Contoh yang salah:

# blok percabangan if

if username == 'petanikode':

print("Selamat Datang Admin")

print("Silahkan ambil tempat duduk")

# blok percabangan for

for i in range(10):

print i

Ada beberapa macam blok program:

Blok Percabangan

Blok Perulangan

Blok Fungsi

Blok Class

Blok Exception

Blok With

28. bantuin ya...apakah arti dari kata sensitif


Sensitif yaitu dapat diartikan sebagai cepat dalam menerima sebuah rangsangan atau peka. Selain itu, dapat diartikan sebagai mudah untuk membangkitkan emosi.

Pembahasan

Sensitif yaitu telah berupa sebuah homonim dikarenakan arti-artinya tersebut telah mempunyai pelafalan dan juga ejaan yang dinilai sama namun mempunyai sebuah makna yang dinilai berbeda.

Arti dari kata sensitif tersebut dapat termasuk kedalam jenis-jenis dari kata kiasan sehingga penggunaan kata sensitif tersebut dapat berarti bahwa bukan merupakan arti yang sesungguhnya.

Sensitif ini telah mempunyai arti dalam kata sifat ataupun kelas adjektifa sehingga kata sensitif tersebut dapat merubah kata ganti atau kata benda yang umumnya telah membuat atau menjelaskannya untuk dapat menjadi lebih spesifik.

33. apa itu peka dan sensitif ​


Penjelasan:Peka tentulah berbeda dengan sensitif. Atau, bisa juga sama? Keduanya bisa juga dianggap sebagai aspek yang saling mempengaruhi. Kepekaan dapat muncul karena adanya perasaan yang sensitif terhadap lingkungan atau fenomena sekitarnya, sedangkan sensitif juga dapat disebabkan karena kepekaan diri terhadap kondisi sekitar dimana kita berinteraksi. Apakah dengan demikian ‘peka’ dan ‘sensitif’ adalah dua hal yang sama, bersifat substitusi, ataukah sabab akibat? Mari berpikir!

Jika dilihat dari segi kebahasaan, ‘peka’ mengandung makna yang lebih positif daripada ‘sensitif’. Seseorang yang mempunyai sifat peka akan cenderung mampu menangkap dan menilai gejala-gejala yang muncul di lingkungan masyarakatnya dan mengambil langkah positif atas fenomena-fenomena di sekitarnya. Orang yang mempunyai kepekaan tinggi, akan cenderung tumbuh menjadi orang yang konstruktif (secara mental). Selanjutnya, ia akan bisa secara langsung atau tidak langsung mengembangkan orang lain disekitarnya. Akan tetapi, ada juga orang yang mempunyai kepekaan tinggi yang tidak mampu menyuarakan perasaannya dan hanya menjadi pengikut dari orang lain yang ia anggap sejalan dan mampu menyuarakan perasaannya yang mempunyai kekekuatan lebih di komunitasnya dibandingkan dirinya.

Lain halnya dengan ‘sensitif’. Seperti pada pendapat sebelumnya, kata ‘sensitif’ cenderung mempunyai makna negatif. Hampir sama seperti ‘peka’, seseorang yang sensitif juga dapat merasakan menangkap gejala-gejala di sekitarnya tetapi yang lebih mengarah/berhubungan dengan dirinya sendiri; bedanya adalah bahwa seorang sensitif kurang mampu menilai fenomena itu secara positif. Seorang sensitif akan cenderung merasa tidak nyaman dengan lingkungannya. Jika sifat sensitif itu tidak dapat dikontrol dan diolah dengan baik, seorang sensitif dapat mengalami tekanan batin/kejiwaan yang dapat mengganggu perkembangan mentalnya. Seorang sensitif harus mencoba untuk terus berinteraksi dengan masyarakat agar ia mampu mengolah sifat sensitif menjadi peka. Akan diperlukan seorang rekan yang mampu mengarahkan seorang sensitif dalam interaksinya di masyarakat, namun perkembangan pribadi bergantung pada diri sendiri.

Apakah mungkin kita mempunyai kedua sifat tersebut? Mungkin! Tetap, pengolahan diri/batin adalah jurus jitu yang diperlukan untuk membentuk diri menjadi lebih positif konstruktif.

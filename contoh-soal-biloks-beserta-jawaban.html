2. coba dong kasi contoh soal biloks sekalian penjelasannya​


1. Tentukan bilangan oksidasi S dalam SO3!

Jawab:

Jumlah bilangan oksidasi SO3 = 0

Jumlah bilangan oksidasi O = -2

Maka:

⇒ b.o S + (3 × b.o O) = 0

⇒ b.o S + (3 × -2) = 0

⇒ b.o S + (-6) = 0

⇒ b.o S = 0 + 6

⇒ b.o S = 6

Jadi, bilangan oksidasi S dalam SO3 adalah 6.

2. Tentukan bilangan oksidasi S dalam SO2!

Jawab:

Jumlah bilangan oksidasi SO2 = 0

Jumlah bilangan oksidasi O = -2

Maka:

⇒ b.o S + (2 × b.o O) = 0

⇒ b.o S + (2 × -2) = 0

⇒ b.o S + (-4) = 0

⇒ b.o S = 0 + 4

⇒ b.o S = 4

Jadi, bilangan oksidasi S dalam SO2 adalah 4.

3. Tentukan biloks F dalam FeO!

Jawab:

Jumlah bilangan oksidasi FeO = 0

Jumlah bilangan oksidasi O = -2

Maka:

⇒ b.o F + (1 × b.o O) = 0

⇒ b.o F + (1 × -2) = 0

⇒ b.o F + (-2) = 0

⇒ b.o F = 0 + 2

⇒ b.o F = 2

Jadi, bilangan oksidasi F adalam FeO adalah 2.

4. Tentukan bilangan oksidasi Fe dalam Fe2O3!

Jawab:

Jumlah bilangan oksidasi Fe2O3 = 0

Jumlah bilangan oksidasi O = -2

Maka:

⇒ (2 × b.o F) + (3 × b.o O) = 0

⇒ (2 × b.o F) + (3 × -2) = 0

⇒ (2 × b.o F) + (-6) = 0

⇒ (2 × b.o F) = 0 + 6

⇒ (2 × b.o F) = 6

⇒ b.o F = 6/2

⇒ b.o F = 3

Jadi, bilangan oksidasi F adalam Fe2O3 adalah 3.

5. Tentukan biloks N dalam NO2!

Jawab:

Jumlah bilangan oksidasi NO2 = 0

Jumlah bilangan oksidasi O = -2

Maka:

⇒ (b.o N) + (2 × b.o O) = 0

⇒ (b.o N) + (2 × -2) = 0

⇒ (b.o N) + (-4) = 0

⇒ b.o N = +4

Jadi, biloks N dalam NO2 = +4.

6. Tentukan bilangan oksidasi Cl adalam ion ClO3−!

Jawab:

Jumlah bilangan oksidasi ClO3− = −1

Bilangan oksidasi O = -2

Maka:

⇒ (b.o Cl) + (3 × b.o O) = −1

⇒ (b.o Cl) + (3 × -2) = −1

⇒ (b.o Cl) + (-6) = −1

⇒ b.o Cl = −1 + 6

⇒ b.o Cl = 5

Jadi, biloks Cl dalam ClO3– = +5.

7. Tentukan bilangan oksidasi N dalam ion NH4+!

Jawab:

Jumlah bilangan oksidasi NH4+ = +1

Jumlah bilangan oksidasi H = +1

Maka:

⇒ (b.o N) + (4 × b.o H) = 1

⇒ (b.o N) + (4 × 1) = 1

⇒ (b.o N) + (4) = 1

⇒ b.o N = 1 – 4

⇒ b.o N = −3

Jadi, biloks N dalam NH4+ = –3.

8. Tentukan biloks S dalam ion SO42–!

Jawab:

Jumlah bilangan oksidasi SO42– = −2

Bilangan oksidasi O = −2

Maka:

⇒ (b.o S) + (4 × b.o O) = −2

⇒ (b.o S) + (4 × −2) = −2

⇒ (b.o S) + (−8) = −2

⇒ b.o S = −2 + 8

⇒ b.o S = 6

Jadi, biloks S dalam SO42– = +6.

9. Tentukan bilangan oksidasi O dalam H2O2!

Jawab:

Jumlah bilangan oksidasi H2O2 = 0

Bilangan oksidasi H = +1

Maka:

⇒ (2 × b.o H) + (2 × b.o O) = 0

⇒ (2 × 1) + (2 × b.o O) = 0

⇒ (2) + (2 × b.o O) = 0

⇒ 2 × b.o O = −2

⇒ b.o O = −2/2

⇒ b.o O = −1

Jadi, bilangan oksidasi O dalam H2O2 adalah -1.

10. Tentukan biloks Mn adalam MnO4−!

Jawab:

Jumlah bilangan oksidasi MnO4− = −1

Bilangan oksidasi O = −2

Maka:

⇒ (b.o Mn) + (4 × b.o O) = −1

⇒ (b.o Mn) + (4 × −2) = −1

⇒ (b.o Mn) + (−8) = −1

⇒ b.o Mn = −1 + 8

⇒ b.o Mn = 7

Jadi, biloks Mn dalam ion MnO4− adalah + 7.

16. Jelaskan apa itu biloks dan contohnya?


boloks adalah angka yang menunjukkan jumlah elektronbiloks didefinisikan sebagai jumlah muatan negatif dan positif dalam atom, yang secara tidak langsung menandakan jumlah elektron yang telah diterima atau diserahkan. Pengertian dari reaksi redoks berdasarkan konsep pelepasan dan penggabungan oksigen adalah reaksi yang mengalami pelepasan dan penerimaan oksigen secara bersamaan. Contoh reaksi redoks berdasarkan konsep pelepasan dan penggabungan oksigen adalah 2 Mg + CO₂ -> 2 MgO + C. Pengertian dari reaksi redoks berdasarkan konsep pelepasan dan penerimaan elektron adalah reaksi yang mengalami pelepasan dan penerimaan elektron secara bersamaan. Contoh reaksi redoks berdasarkan konsep pelepasan dan penerimaan elektron adalah Zn + Cu²⁺ → Cu + Zn²⁺ . Dimana Zn mengalami oksidasi ( karena melepaskan elektron) dan Cu²⁺ mengalami reduksi ( karena menerima elektron).

Pengertian dari reaksi redoks berdasarkan konsep perubahan harga bilangan oksidasi  adalah reaksi yang mengalami peningkatan dan penurunan bilangan oksidasi secara bersamaan. Contoh reaksi redoks berdasarkan konsep perubahan harga bilangan oksidasi adalah Cr₂O₃ + 2 Al ⇒ 2 Cr + AlO₃ . Dimana Al mengalami oksidasi ( karena peningakatan bilangan oksidasi) dan Cr₂O₃ mengalami reduksi ( karena Cr mengalami penurunan bilangan oksidasi).

24. soal materi tentang biloks


Soal No. 1
Tentukan bilangan oksidasi unsur S dalam senyawa H2SO4

Pembahasan
H2SO4 jika diuraikan sesuai ion-ion pembentuknya adalah:
H2SO4 → 2H+ + SO42−
Yang ditanyakan adalah biloks S, sehingga cukup melihat pada ion SO42− saja. Jumlah biloks pada ion ini harus − 2. Misal biloks S adalah x dan diingat biloks O disini adalah −2 sehingga:

x = −2 + 8 = +6
Jadi biloks S pada senyawa di atas adalah +6

38. Tentukan biloks dari umsur: Tentukan biloks Mg dari Mg3(PO4)2 biloks Cu dari CuCl2 dan biloks C dari CaCO3 Tolong dijawab ya


Biloks dari unsur berikut :  

1.  Biloks Mg dari Mg₃(PO₄)₂ adalah +2.

2.  Biloks Cu dari CuCl₂ adalah +2.

3.  Biloks C dari CaCO₃ adalah +4.

Pembahasan :1. Biloks Mg dari Mg₃(PO₄)₂

biloks P = +5

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.  

biloks Mg₃(PO₄)₂ = 0  

(3 × biloks Mg) + (2 × biloks P) + (8 × biloks O) = 0  

(3 × biloks Mg) + (2 × (+5)) + (8 × (-2)) = 0  

(3 × biloks Mg) + (+10) + (-16) = 0  

(3 × biloks Mg) + (-6) = 0  

(3 × biloks Mg) = +6

Biloks Mg = [tex]\frac{+6}{3}[/tex]

Biloks Mg = +2

2. Biloks Cu dari CuCl₂

biloks Cl = -1 (Golongan VIIA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks CuCl₂ = 0  

biloks Cu + (2 × biloks Cl) = 0  

biloks Cu + (2 × (-1)) = 0  

biloks Cu + (-2) = 0  

biloks Cu = +2

3. Biloks C dari CaCO₃

biloks Ca = +2 (merupakan unsur golongan IIA)  

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.  

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks CaCO₃ = 0  

biloks Ca + biloks C + (3 × biloks O) = 0  

(+2) + biloks C + (3 × (-2)) = 0  

(+2) + biloks C + (-6) = 0  

biloks C + (-4) = 0  

biloks C = +4

⇒ Kesimpulan, maka biloks dari unsur berikut :  

1.  Biloks Mg dari Mg₃(PO₄)₂ adalah +2.

2.  Biloks Cu dari CuCl₂ adalah +2.

3.  Biloks C dari CaCO₃ adalah +4.

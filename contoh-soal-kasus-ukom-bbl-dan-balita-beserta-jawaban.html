21. C. LATIHAN SOAL: 1. PT. Sabar memproduksi mainan anak, biaya yang dikeluarkan dikelompokkan menjadi: Biaya utama Biaya Konversi Biaya Produksi Hitunglah BBL, BTKL dan BOP? Penyelesaian: a) Biaya produksi = BBL + BTKL + BOP Biaya konversi = BTKL + BOP BBL b) Biaya utama c) Biaya konversi Pembuktian: BBL BTKL BOP BP Rp. 1.000.000,- Rp. 1.500.000,- Rp. 2.100.000,- = BBL + BTKL BBL = BTKL BTKL + BOP BTKL BOP - = Biaya Bahan Langsung = Biaya Tenaga Kerja Langsung = Biaya Overhead Pabrik = = Biaya Produksi = Rp. 2.100.000,- = Rp. 1.500.000.- (-) = Rp. 600.000.- = Rp. 1.000.000,- = Rp. 600.000,- (-) =Rp. 400.000.- = = Rp. 1.500.000,- =Rp. 400.000.- (-) =Rp. 1.100.000,- = Rp. 600.000,- = Rp. 400.000,- = Rp. 1.100.000,- (+) = Rp. 2.100.000.-​


Jawaban:

Untuk menghitung BBL (Biaya Bahan Langsung), BTKL (Biaya Tenaga Kerja Langsung), dan BOP (Biaya Overhead Pabrik), kita dapat menggunakan rumus sebagai berikut:

Biaya produksi = BBL + BTKL + BOP

Biaya konversi = BTKL + BOP

Biaya utama = BBL

Diketahui:

Biaya produksi = Rp. 2.100.000,-

Biaya konversi = Rp. 1.500.000,-

Biaya utama = Rp. 1.000.000,-

a) BBL

Biaya utama = BBL

BBL = Rp. 1.000.000,-

b) BTKL

Biaya konversi = BTKL + BOP

BTKL = Biaya konversi - BOP

BTKL = Rp. 1.500.000,- - Rp. 600.000,-

BTKL = Rp. 900.000,-

c) BOP

Biaya konversi = BTKL + BOP

BOP = Biaya konversi - BTKL

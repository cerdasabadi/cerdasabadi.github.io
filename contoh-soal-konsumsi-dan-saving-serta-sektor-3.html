2. Fungsi konsumsi c=20+0,75y.Hitunglah fungsi saving dan jumlah saving


Jawaban:

fungsi saving adalah S = Y - 20 - 0,75y, dan jumlah saving untuk pendapatan sebesar 100 adalah 5.

Penjelasan:

Fungsi konsumsi yang diberikan adalah c = 20 + 0,75y, di mana y adalah pendapatan. Untuk mencari fungsi saving, kita dapat menggunakan persamaan:

S = Y - C

di mana S adalah saving (tabungan) dan Y adalah pendapatan. Dalam hal ini, karena tidak diberikan nilai konsumsi, kita dapat mengganti nilai C dengan fungsi konsumsi yang diberikan. Sehingga:

S = Y - (20 + 0,75y)

S = Y - 20 - 0,75y

Untuk mencari jumlah saving (tabungan), kita dapat mengganti nilai Y dengan jumlah pendapatan yang diberikan. Misalnya, jika jumlah pendapatan (Y) adalah 100, maka:

S = 100 - 20 - 0,75(100)

S = 100 - 20 - 75

S = 5

Jadi, fungsi saving adalah S = Y - 20 - 0,75y, dan jumlah saving untuk pendapatan sebesar 100 adalah 5.

4. 3. Diketahui : C = 0,75 Yd + 25 triliun rp Tx = 30 triliun rp Tr = 50 triliun per tahun Pertanyaan: a. Carilah fungsi konsumsi dan juga fungsi saving setelah adanya pajak dan sebelum adanya transfer pemerintah ! b. Carilah fungsi konsumsi dan juga fungsi saving sebelum adanya pajak dan setelah adanya transfer pemerintah!​


a. Fungsi konsumsi setelah adanya pajak dan sebelum adanya transfer pemerintah dapat ditentukan dengan mengurangi pajak dari pendapatan nasional. Pendapatan nasional dapat ditentukan dengan menambah konsumsi dengan investasi, pajak, dan transfer pemerintah.

C = 0,75 Yd + 25 triliun rp

Yd = C + Tx + Tr

Yd = C + 30 triliun rp + 0

Fungsi saving setelah adanya pajak dan sebelum adanya transfer pemerintah dapat ditentukan dengan mengurangi konsumsi dari pendapatan nasional.

S = Yd - C

S = (C + Tx) - C

S = Tx = 30 triliun rp

b. Fungsi konsumsi sebelum adanya pajak dan setelah adanya transfer pemerintah dapat ditentukan dengan menambah transfer pemerintah dari pendapatan nasional.

C = 0,75 Yd + 25 triliun rp + Tr

Yd = C + Tx

Yd = C + 30 triliun rp

Fungsi saving sebelum adanya pajak dan setelah adanya transfer pemerintah dapat ditentukan dengan mengurangi konsumsi dari pendapatan nasional.

S = Yd - C

S = (C + Tx + Tr) - (C + Tr)

S = Tx = 30 triliun rp

Secara umum fungsi konsumsi dan saving ditentukan oleh pendapatan nasional, jika pajak dan transfer pemerintah berubah maka akan mempengaruhi fungsi tersebut

20. Fungsi Konsumsi C = 50 + 0,75YFungsi investasi I = 35Hitunglah pendapatan nasional keaeimbangan Hitunglah konsumsi keseimbangan dan saving keseimbangan​


Jawaban:

1. Untuk mencari pendapatan nasional keaeimbangan, pertama-tama kita perlu mencari pendapatan nasional yang sesuai dengan fungsi konsumsi dan fungsi investasi yang diberikan. Pendapatan nasional dapat dicari dengan menambahkan konsumsi dan investasi:

Pendapatan nasional = Konsumsi + Investasi

Kita dapat mencari konsumsi dengan menggunakan fungsi konsumsi yang diberikan, yaitu C = 50 + 0,75Y. Kita juga dapat mencari investasi dengan menggunakan fungsi investasi yang diberikan, yaitu I = 35.

Jadi, pendapatan nasional dapat dicari dengan persamaan berikut:

Pendapatan nasional = C + I

= 50 + 0,75Y + 35

= 85 + 0,75Y

2. Untuk mencari konsumsi keseimbangan dan saving keseimbangan, pertama-tama kita perlu mencari pendapatan nasional keaeimbangan dengan menggunakan persamaan yang telah kita buat sebelumnya, yaitu:

Pendapatan nasional = 85 + 0,75Y

Kemudian, kita dapat mencari konsumsi keseimbangan dengan menggunakan fungsi konsumsi yang diberikan, yaitu C = 50 + 0,75Y. Jadi, konsumsi keseimbangan dapat dicari dengan persamaan berikut:

Konsumsi keseimbangan = 50 + 0,75Y

Sedangkan, saving keseimbangan dapat dicari dengan mengurangi konsumsi keseimbangan dari pendapatan nasional keaeimbangan:

Saving keseimbangan = Pendapatan nasional - Konsumsi keseimbangan

= (85 + 0,75Y) - (50 + 0,75Y)

= 35

Jawaban:

Pendapatan nasional keaeimbangan (Y) adalah jumlah pendapatan yang diterima oleh masyarakat dalam suatu negara pada saat pendapatan nasional mencapai titik keaeimbangan. Titik keaeimbangan terjadi ketika konsumsi sama dengan pendapatan nasional, yaitu C = Y.

Untuk mencari pendapatan nasional keaeimbangan, kita harus menyelesaikan persamaan C = Y. Persamaan tersebut dapat ditulis sebagai berikut:

50 + 0,75Y = Y

Dengan mengurangi Y dari kedua sisi persamaan, maka kita dapat menyelesaikannya menjadi:

50 = 0,25Y

Dengan membagi kedua sisi persamaan dengan 0,25, maka kita dapat menemukan nilai Y:

Y = 50 / 0,25

Y = 200

Jadi, pendapatan nasional keaeimbangan adalah 200.

Konsumsi keseimbangan (C) adalah jumlah konsumsi yang terjadi pada saat pendapatan nasional mencapai titik keaeimbangan. Kita sudah mengetahui bahwa pendapatan nasional keaeimbangan adalah 200, jadi kita dapat menggunakan rumus C = 50 + 0,75Y untuk mencari konsumsi keseimbangan.

C = 50 + 0,75Y

C = 50 + 0,75 * 200

C = 50 + 150

C = 200

Jadi, konsumsi keseimbangan adalah 200.

Saving keseimbangan adalah selisih antara pendapatan nasional dan konsumsi. Jadi, untuk mencari saving keseimbangan, kita dapat mengurangi konsumsi keseimbangan dari pendapatan nasional keaeimbangan:

Saving keseimbangan = Y - C

Saving keseimbangan = 200 - 200

Saving keseimbangan = 0

Jadi, saving keseimbangan adalah 0.

23. sebutkan contoh kegiatan masyarakat yang bergerak di sektor distribusi dan konsumsi!


Kegiatan usaha berdasarkan bidangnya dapat dibagi dalam beberapa jenis, diantaranya adalah :
1. Usaha Ekstraktif
Usaha ekstraktif merupakan usaha yang dilakukan dengan cara eksplorasi sumber daya alam tanpa mengubah wujud barang produksi tersebut. Contohnya, tambang pasir, emas, batu bara, minyak bumi, dan perak. Tujuannya untuk memenuhi kebutuhan masyarakat akan sumber daya alam.
2. Usaha Jasa.
Usaha jasa merupakan kegiatan produksi alat pemenuhan kebutuhan berupa jasa tertentu. Adapun hasil produksi dari jasa tidak dapat dilihat maupun diraba seperti barang, melainkan hanya dapat dirasakan berupa kepuasan konsumen akan pelayanan jasa tersebut. Contoh produksi jasa antara lain : perawatan kecantikan, asuransi, penginapan, pariwisata, telekomunikasi, dan aneka produk jasa lainnya.
3. Usaha Agraris.
Negara Indonesia merupakan negara agraris yang terdiri dari banyak daratan dengan usaha-usaha pertanian. Adapun usaha agraris, yaitu usaha yang bergerak dengan cara mengelola tanah dan hewan untuk menghasilkan alat pemenuhan kebutuhan manusia. Hasil produksi agraris misalnya beras, jagung, kedelai, dan berbagai ternak yang menjadi bahan pokok manusia untuk memperoleh makanan. Contoh dari produksi agraris, yaitu perkebunan, pertanian, perikanan, dan peternakan.
4. Usaha Perdagangan.
Usha perdagagan merupakan usaha mencari keuntungan dengan cara menjualbelikan barang dagangan tanpa mengubah wujud dari barang dagang tersebut. Dengan kata lain tidak terjadi proses produksi dari bahan mentah ke barang jadi. Pedagang hanya memperjualbelikan barang yang telah diproduksi oleh produsen. Keuntungan perdagangan diproleh dari selisih harga beli dengan harga jual barang. Bentuk perdagangan sering kamu lihat seperti kios, warung, supermarket, dan pusat perniagaan (mall).
5. Usaha Industri
Usaha industri berupa kegiatan produksi yang menghasilkan barang maupun jasa dengna cara mengolah bahan mentah yang kemudian diproses menjadi bahan jadi. Adapun contoh dari industri seperti ini seperti pabrik sepatu, makanan dan minuman kaleng serta tekstil.

C = 100 + 0,75 y

Jawab

Langkah 1: menentukan fungsi saving

C = a + b Y

S = a + (1-b) Y

S = 100 + (1 - 0,75) Y

S = 100 + 0,25 Y

Langkah 2: menentukan nilai national income equilibrium

Ye = [tex]\frac{co + cTx + I + G}{1 - c}[/tex]

Ye = [tex]\frac{100 + 0,75(100) + 0 + 0}{1 - 0,75}[/tex]

Ye = [tex]\frac{175}{0,25}[/tex]

Ye = 700

Langkah 3: menentukan Ye jika I = 10

Ye = [tex]\frac{co + cTx + I + G}{1 - c}[/tex]

Ye = [tex]\frac{100 + 0,75(100) + 10 + 0}{1 - 0,75}[/tex]

Ye = [tex]\frac{185}{0,25}[/tex]

Ye = 740

Langkah 4: menentukan kenaikan konsumsi dan saving setelah ada investasi

C = cYe + co

C = 0,75 (740) + 100

C = 555 + 100

C = 655

S = Ye - C

S = 740 - 655

S = 85

30. Diketahui fungsi konsumsi suatu Negara adalah : C = 60 + 0,70 Y dan investasi (I) adalah 25. a. Berapa kecendrungan konsumsi (MPC) dan saving (MPS) dari Negara tersebut ? b Hitunglah pendapatan nasional keseimbangan 2 sektor dengan pendekatan pengeluaran dan pendekatan injeksi kebocoran (Saving). Berapa APC? c. Apabila investasi Negara tersebut naik menjadi 40, berapa pendapatan keseimbangan baru dan berapa multiplier effect dari investasi tersebut ?


Diketahui :

C = 60 + 0,70Y   ⇔  S = -60 + 0,30Y

I = 25

Penyelesaian :

a. MPC = 0,70  ↔ Didapat dari C = 60 + 0,70Y.

MPS = 0,30  ↔ Didapat dari S = -60 + 0,30Y.

b. Menghitung keseimbangan pendapatan nasional 2 sektor;

Pendekatan Pengeluaran

Rumus :  Y = C + I

→ Y = 60 + 0,70Y + 25

Y - 0,7Y = 60 + 25

0,3Y = 85

Y = 283,3.

Pendekatan Injeksi Kebocoran

Rumus :  S = I

→ -60 + 0,30Y = 25

0,3Y = 85

Y = 85/0,3

Y = 283,3.

Nilai APC

Rumus menghitung APC = [tex]\frac{C}{Yd}[/tex] .

Y = 283,3

C = 60 + 0,70 (283,3) = 60 + 198,31 = 258,31.

APC = [tex]\frac{258,31}{283,3}[/tex] = 0.9117.

c. Menghitung ΔY (perubahan pendapatan nasional)

Bila ΔI = 40

ΔY = ΔI  x Multiplier

ΔY = ΔI  x  [tex]\frac{1}{1 - MPC}[/tex]

ΔY = ΔI  x  [tex]\frac{1}{MPS}[/tex]

ΔY = 40 x  [tex]\frac{1}{0,30}[/tex]

ΔY = 133,3.

Sehingga, Y' = C + I + ΔY

Y ' = 258,31 + 25 + 133,3

Y ' = 416,61.

Multiplier Effect dari Investasi =  [tex]\frac{1}{1 - MPC}[/tex]  atau  [tex]\frac{1}{MPS}[/tex].

6. Invers dari matriks identitas adalah..a. Matriks Inversb. Matriks Nolc. Matriks satud. Matriks Identitas​


Jawab:

JANGAN LUPA BINTANG 5 NYA YA KAK

Penjelasan dengan langkah-langkah:

A, matriks identitas.

Sebuah matriks dikatakan singular (tidak berinvers) jika determinannya 0. Matriks identitas adalah matriks persegi yang digunakan untuk menyatakan transformasi identitas. Matriks identitas berukuran n x n dengan elemen diagonalnya bernilai 1 dan elemen non-diagonalnya bernilai 0.

Contoh:

Matriks identitas 2 x 2:

1 0

0 1

Matriks identitas 3 x 3:

1 0 0

0 1 0

0 0 1

Matriks identitas mempunyai invers yang sama dengan matriks itu sendiri. Jadi invers dari matriks identitas adalah matriks identitas itu sendiri.

Pilihan A adalah "Matriks Identitas" sehingga jawabannya adalah A.

Pilihan B "Matriks Nol" dan C "Matriks Satu" bukan merupakan invers dari matriks identitas.

Pilihan D "Matriks Identitas" adalah matriks itu sendiri, bukan inversnya.

Jadi jawaban yang benar adalah A, "Matriks Identitas".

PEMBAHASAN

Matriks adalah suatu konsep matematika yang memaparkan tentang bentuk penyusunan bilangan ke dalam suatu kolom ataupun baris yang dibatasi oleh [ ... ] atau ( ... ). Matriks ini terdiri dalam bermacam kriteria, diantaranya:

Matriks nol:

(semua elemennya adalah angka nol).

[tex]\left[\begin{array}{cc}\sf 0&\sf 0\\\sf 0&\sf 0\end{array}\right] [/tex] (dua kolom)

[tex]\left[\begin{array}{ccc}\sf 0&\sf 0&\sf 0\\\sf 0&\sf 0&\sf0\\\sf0&\sf 0&\sf 0\end{array}\right] [/tex] (tiga kolom).

Matriks baris:

(disusun dengan konsep baris).

[tex]\left[\begin{array}{cc}\sf a&\sf b &\sf c\end{array}\right] [/tex]

Matriks kolom:

(disusun dengan konsep kolom).

[tex]\left[\begin{array}{ccc}\sf a\\\sf b \\\sf c\end{array}\right] [/tex]

Matriks diagonal:

(elemen anti 0 hanya ada pada bagian diagonal).

[tex]\left[\begin{array}{ccc}\sf a&\sf 0&\sf 0\\\sf 0&\sf b&\sf0\\\sf0&\sf 0&\sf c\end{array}\right] [/tex]

Matriks persegi:

(berbentuk persegi).

[tex]\left[\begin{array}{cc}\sf a&\sf b\\\sf c&\sf d\end{array}\right] [/tex]

Matriks segitiga:

(salah satu baris/kolom elemennya adalah bilangan berselisih 3).

[tex]\left[\begin{array}{ccc}\sf 1&\sf 4&\sf 7\\\sf a&\sf b&\sf c\\\sf d&\sf e&\sf f\end{array}\right] [/tex]

Matriks identitas:

(Matriks diagonal yang salah satu diagonalnya memiliki urutan runtut angka 1).

(elemen anti 0 hanya ada pada bagian diagonal).

[tex]\left[\begin{array}{ccc}\sf 1&\sf 0&\sf 0\\\sf 0&\sf 1&\sf0\\\sf0&\sf 0&\sf 1\end{array}\right] [/tex]

Buktikan bahwa [tex]\displaystyle \frac{\tan x+\sec x-1}{\tan x-\sec x+1}=\frac{1+\sin x}{\cos x}[/tex]

Penjelasan dengan langkah-langkah:

Berdasarkan identitas Pythagoras tan² x + 1 = sec² x

[tex]\begin{aligned}\frac{\tan x+\sec x-1}{\tan x-\sec x+1}&\:=\frac{\tan x+\sec x-(\sec^2 x-\tan^2 x)}{\tan x-\sec x+1}\\\:&=\frac{\tan x+\sec x-(\sec x+\tan x)(\sec x-\tan x)}{\tan x-\sec x+1}\\\:&=\frac{(\tan x+\sec x)[1-(\sec x-\tan x)]}{\tan x-\sec x+1}\\\:&=\tan x+\sec x\\\:&=\frac{\sin x}{\cos x}+\frac{1}{\cos x}\\\:&=\frac{1+\sin x}{\cos x}\end{aligned}[/tex]

25. Tuliskan contoh soal identitas trigonometri, jawabannya dan pembahasannya.​


Diketahui :

Pembuktian suatu identitas trigonometri

Ditanya :

Contoh soal pembuktian identitas trigonometri ... ?

Jawab :

1. Soal : Buktikan (sin 2x)/sin x = (1 + cos 2x)/cos x

Penyelesaian :

Pembuktian dari kiri dan kanan langsung.

[tex]\frac{sin2x}{sinx} = \frac{1+cos2x}{cosx}\\\frac{2.sinx.cosx}{sinx} = \frac{(sin^2x+cos^2x)+(cos^2x - sin^2x)}{cosx}\\2.cosx = \frac{2.cos^2x}{cosx}\\2.cosx = 2cosx[/tex]

Terbukti bahwa (sin 2x)/sin x = (1 + cos 2x)/cos x adalah benar.

2. Soal : Buktikan (1 - cos 2x)/(1 - cos² x) = 2

Penyelesaian :

Pembuktian dari kiri.

[tex]\frac{1-cos2x}{1-cos^2x} = 2\\\frac{(sin^2x +cos^2x)-(cos^2x - sin^2x)}{sin^2x} = 2\\\frac{2sin^2x}{sin^2x} = 2\\2 = 2[/tex]

Terbukti bahwa (1 - cos 2x)/(1 - cos² x) = 2 adalah benar.

3. Soal : Buktikan cosec 2x = (1 + cot² x)/(2.cot x)

Penyelesaian :

Pembuktian dari kanan.

[tex]cosec2x = \frac{1+cot^2x}{2.cotx}\\cosec2x = \frac{\frac{sin^2x}{sin^2x}+\frac{cos^2x}{sin^2x}}{2.\frac{cosx}{sinx}}\\cosec2x = \frac{\frac{sin^2x+cos^2x}{sin^2x}}{\frac{2.cosx}{sinx}}\\cosec2x = \frac{\frac{1}{sin^2x}}{\frac{2.cosx}{sinx}}\\cosec2x = \frac{1}{sin^2x} . \frac{sinx}{2.cosx}\\cosec2x = \frac{1}{2.sinx.cosx}\\cosec2x = \frac{1}{sin2x}\\cosec2x = cosec2x[/tex]

Terbukti bahwa cosec 2x = (1 + cot² x)/(2.cot x) adalah benar.

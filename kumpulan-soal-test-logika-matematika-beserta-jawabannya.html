f(x) = -x² - 4x

Penjelasan dengan langkah-langkah:

titik puncak P(-2, 4)

p = -2

q = 4

y = a(x - p)² + q

y = a(x - (-2))² + 4

y = a(x + 2)² + 4

melalui titik (0, 0)

x = 0

y = 0

y = a(x + 2)² + 4

0 = a(0 + 2)² + 4

0 = 4a + 4

4a = -4

a = -1

Persamaan kuadrat

y = a(x + 2)² + 4

y = -1(x² + 4x + 4) + 4

y = -x² - 4x - 4 + 4

y = -x² - 4x

f(x) = -x² - 4x

15. (SOAL PELUANG) Pada tes fisika terdapat 5 soal pilihan ganda dan 3 soal essai. Peserta test harus mengerjakan 3 soal pilihan ganda dan  2 soal essai. Banyaknya pilihan untuk mengerjakan test adalah.... cara


Pilihan ganda ada 5 dikerjakan 3 maka ada 5C3 = 10 cara
Esai ada 3 dikerjakan 2 maka ada 3C2 = 3 cara
Jadi, total cara pengerjaan ada 10 . 3 = 30 caradik : 5 soal pilihan ganda
       3 soal essai
      harus mengerjakan 3 soal pilihan ganda dan 2 soal esai
dit : banyak cara untuk mengerjakan test adalah ..?

menggunakan metode kombinasi :
Pilihan berganda : C(5,3) = 5 ! / (5-3) ! . 3 ! = 5.4.3.2! / 2! 3! = 60 / 3.2.1 = 10 cara
Essai : C(3,2) = 3! / (3-2) ! 2! = 3 . 2! / 1 ! 2! = 3/1 = 3 cara

sehingga total cara = 10 + 3 = 13 cara

18. Dalam suatu test matematika jika menjawab benar di beri skor 4 jika slah di beri skor -2 jika tidak menjawab di beri skor -1. Dari 25 soal Sinta menjawab dengan benar 18 soal dan menjawab salah 3 soal, berapakah skor yang di peroleh Sinta


Jawaban:

jawaban nya adalah 62

Penjelasan dengan langkah-langkah:

diket :

25 soal = benar (18), salah (3)

tidak dijawab (25-(18+3)= 25-21= 4)

skor = benar(4), salah(-2), tidak dijawab(-1)

ditanya :

skor yang diperoleh Sinta

jawab :

skor jawab benar = 18×4 = 72

skor jawab salah = 3×(-2) = (-6)

skor tidak dijawab = 4×(-1) = (-4)

SKOR TOTAL

= skor jawab benar + skor jawab salah + skor tidak dijawab

= 72 + (-6) + (-4)

= 72 - 6 - 4

= 62

jadi skor yang diperoleh Sinta adalah 62

Dari 50 soal, 40 soal dijawab benar dengan nilai 70
Misal soal benar adalah b, soal salah adalah s, tidak dijawab adalah t
Maka,

Nilai anak = (Soal benar x 2) + (Soal salah x -1) + (Tidak dijawab x 0)
70 = (40 x 2) + (s x -1) + (t x 0)
70 = 80 + (-s) + 0
70 = 80 - s
70 - 80 = -s
-10 = -s
s = 10

Jadi, soal yang dijawab salah adalah 10 soal

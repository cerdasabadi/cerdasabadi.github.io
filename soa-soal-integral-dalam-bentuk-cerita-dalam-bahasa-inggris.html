20. #Soal Integral !!!!!!


Penyelesaian:

∫ (6x - 12) √(x^2 - 4x + 8) dx

misal

u = x^2 - 4x + 8

du = 2x - 4

dx = du/(2x - 4)

∫ (6x - 12) . u^1/2 . du/(2x - 4)

3 . ∫ . u^1/2 + C

= 3 . 2/3 . u^3/2 + C

= 2 . u^3/2 + C

= 2 (x^2 - 4x + 8)^3/2 + C

====================

Detil Jawaban

Kelas: 11

Mapel: Matematika

Bab: Integral Tak Tentu

Kode: 11.2.10

Kata Kunci: integral

Jawab:

[tex]\int\limits {(6x-12)\sqrt{ x^{2}-4x+8 } } \, dx \\=\int\limits {(6x-12)\sqrt{ x^{2}-4x+8 } } \, \frac{d( x^{2}-4x+8)}{(2x-4)} \\=\int\limits {3(2x-4)\sqrt{ x^{2}-4x+8 } } \, \frac{d( x^{2}-4x+8)}{(2x-4)}\\=\int\limits {3\sqrt{ x^{2}-4x+8 } } \,} \, d(x^{2}-4x+8)\\= \int\limits {3(x^{2}-4x+8) ^{ \frac{1}{2} } } } \,} \, d(x^{2}-4x+8)\\=\frac{3}{ \frac{1}{2} +1} (x^{2}-4x+8) ^{ \frac{1}{2}+1 } +c\\=\frac{3}{ \frac{3}{2} } (x^{2}-4x+8) ^{ \frac{3}{2} } +c\\=2 (x^{2}-4x+8) ^{ \frac{3}{2} }+c\\[/tex]

39. pleasee help. cari bentuk integralnya ini soal TO DKI kmaren.

coba aja masukin, kalo x= 0, y atau f(x) nya harus +1. kalo y=0, Xnya harus -2

(-2, 0) dan (0, 1)

persamaan garisnya :

y - y1 / y2 - y1 = x - x1 / x2 - x1

y - 0 / 1 - 0 = x - (-2) / 0 - (-2)
y / 1 = x + 2 / 2
x + 2 = 2y
2y = x + 2
y = 1/2x + 1

Batas atas = 3 dan batas bawah = -2

Maka jawabannya adalah B.

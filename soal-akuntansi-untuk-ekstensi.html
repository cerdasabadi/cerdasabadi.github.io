2. Dalam persamaan akuntansi ekstensi perspektif sejarah , persamaannya adalah : asset + beban = liabilitis + ekuitas + penghasilan . Terangkan arti masing-masing elemn dan berilah tiga contoh akun untuk masing-masing elemen .


Jawaban:

- Aset, merupakan sumber daya ekonomi yang digunakan perusahaan untuk menjalankan operasional perusahaan. Contoh: Kas, Piutang, Persediaan

- Beban, merupakan pengorbanan atas sumber daya yang telah digunakan untuk menjalankan operasional perusahaan. Contoh: Beban sewa, beban perlengkapan, beban listrik.

- Liabilitas, merupakan klaim dari kreditor atas aset dari perusahaan. Contoh: utang gaji, utang listrik, utang sewa

- Ekuitas, merupakan sisa klaim pemilik perusahaan atas aset perusahaan. Contoh: modal disetor, modal saham, laba ditahan.

- Penghasilan, merupakan perolehan atas penyerahan barang/jasa kepada pelanggan. Contoh: pendapatan jasa bengkel,penjualan,pendapatan jasa konsultasi.

14. tolong beri soal analisis untuk persamaan dasar akuntansi dong


Penjelasan:

Persamaan Dasar Akuntansi – Akuntansi berasal dari kata accounting, yang merupakan ilmu menghitung serta mencatat transaksi yang ada hubungannya dengan keuangan perusahaan. Akuntansi memang penting dilakukan sebagai metode pencatatan transaksi keuangan perusahaan agar nantinya tidak ada penyalahgunaan dana yang dianggarkan.

Begitu pentingnya akuntansi sehingga subjek ini dijadikan mata pembelajaran di sekolah menengah atas, sekolah menengah kejuruan, hingga perguruan tinggi sekalipun. Hal ini dikarenakan ada banyak materi tentang akuntansi yang harus dipelajari agar nantinya bisa mempraktekkan ilmunya dengan benar.

Salah satu materi akuntansi yang penting untuk dipelajari adalah persamaan dasar akuntansi. Persamaan dasar ini bisa memberikan gambaran besar tentang kekayaan perusahaan, hutang, dan modal perusahaan selama periode akuntansi tertentu. Selain itu, persamaan dasar akuntansi juga digunakan untuk melakukan pembukuan sederhana perusahaan.

20. ekstensi file untuk program database adalah


Ekstensi file untuk program database adalah .dbf yaitu File database. Selain itu, ekstensi lain untuk files database adalah .pdb (program database file), .wdb (microsoft work database file), .db (file database), .mdb (microsoft Access Database File).

Pembahasan

Database atau disebut juga basis data yaitu kumpulan data yang berhubungan dengan suatu objek, topik atau tujuan khusus tertentu, biasanya berupa kumpulan dari tabel.

Berikut ini jenis-jenis aplikasi untuk database, yaitu:

Dbase, yaitu aplikasi database yang dikembangkan oleh perusahaan Ashton Tate, untuk platform Apple dan Unix kemudian digunakan dibawah DOS. Ekstensi filenya “dbf” masih dipakai untuk Arcview dan Foxpro.IBM DB2, yaitu aplikasi database pertama menggunakan bahasa SQL. Microsoft SQL Server, yaitu aplikasi database buatan Microsoft, yang berguna untuk database skala besar dan terpusat ke server.My SQL, yaitu aplikasi database yang bersifat gratis (freeware). Digunakan untuk aplikasi berbasis web dan dikombinasikan dengan bahasa pemrograman berbasis web, yaitu PHP.Oracle, yaitu aplikasi database client-server yang digunakan untuk penyimpanan dan pengelolaan data yang sangat besar dan banyak.Microsoft Access, yaitu aplikasi database buatan Microsoft, untuk penggunaan skala kecil dengan file yang dihasilkan berbentuk file accdb (access database).

( Rp 12.000.000 - (Rp. 12.000.000 / 12 × 9))

(9 = April – Desember, Maret tidak bisa dihitung 1 hulan karena sudah berada di akhir tanggal)

b. Beban Gaji Rp 800.000

Utang Beban Rp 800.000

( 8 × Rp. 100.000)

c. Beban Penyusutan / Beban Utilitas Rp5.000.000

Akum. Peny. Mesin Rp5.000.000

(Rp 50.000.000 × 10%)

d. Kas Rp 880.000

Pendapatan Bunga Rp 880.000

( Rp 44.000.000 × 24% × 1/12)

e. Beban Bunga / Beban Utiliti Rp 7.000.000

Utang Beban Rp 7.000.000

( Rp 50.000.000 × 2% × 7)

f. Beban Perlengkapan / Beban Utilitas Rp 1.000.000

Perlengkapan Rp 1000.000

Note : Beban utilitas dipakai apabila tidak ada akun yg bersangkutan

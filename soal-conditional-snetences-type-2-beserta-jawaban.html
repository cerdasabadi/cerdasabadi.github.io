4. apa perbedaan conditional type 1 and type 2


Answer:
perbedaan conditional sentence type 1 dan 2 :

Kalau conditional sentence type 1
Apa yang diinginkan atau diharapkan kemungkinan masih bisa terjadi atau terwujud karena kenyataannya baru akan terjadi dimasa yang akan datang.
Contoh:
They will come to your office if they come here before 9 O'clock
(Mereka akan datang ke kantormu jika mereka datang kesini sebelum pukul sembilan)
Jadi masih ada kemungkinan mereka akan datang karena mereka datang jam 7 atau jam 8.

Sedangkan
Conditional sentence type 2
Kemungkinannya sulit terwujud karena kenyataannya diwaktu sekarang atau saat ini tidak sesuai dengan apa yang kita harapkan atau apa yang kita inginkan.
Contoh:
If i were a rich man, i would buy a new car
(Jika saya orang kaya,saya akan membeli sebuah Mobil Baru)
Kenyataannya:
Saath ini saya bukan orang kaya,Jadi saya tidak membeli sebuah Mobil baru)

Contoh soal conditional sentence type 1, 2 dan 3 yaitu

Change the verb in parentheses to first, second, and third conditional.

We (build) _______ a new house, if we (have) _______ enough money.

Answer:

Type 1: We will build a new house, if we have enough money.

Type 2: I would build a new house, if we had enough money

Type 3: I would have bought a new house, if we had had enough money.

PENJELASAN

=> CONDITIONAL SENTENCE TYPE 1

First conditional sentence merupakan bentuk kalimat pengandaian yang digunakan ketika hasil atau konsekuensi memiliki kemungkinan terjadi di masa depan. Maka dari itu, bentuk kalimat dari conditional sentence tipe pertama ini memiliki bentuk kalimat simple future alias kalimat yang akan datang. Rumus dari kalimat ini adalah: if + simple present, simple future “will”.

=> CONDITIONAL SENTENCE TYPE 2

Second conditional sentences merupakan tipe kalimat pengandaian yang digunakan ketika hasil atau konsekuensi hanya memiliki harapan terwujud walaupun kemungkinannya sangat kecil. Dalam bentuk kalimat ini, rumus yang digunakan adalah if + simple past/were, would/could/might + bare infinitive. Penggunaan would/could/might berfungsi untuk menjelaskan seberapa besar sebuah kondisi itu akan terjadi.

=> CONDITIONAL SENTENCES TYPE 3

Third conditional sentence merupakan sebuah kalimat pengandaian ketika sebuah kondisi tidak mungkin terwujud sama sekali. Dalam tipe kalimat pengandaian yang ketiga ini, bentuk kalimat menggunaka past perfect yang dilengkapi dengan modal auxiliary seperti would, could, dan should. Rumus yang digunakan adalah if + past perfect, would/should/could/might + have + past participle.

19. buatlah 5 conditional sentences type 2 beserta fakta​


Jawaban:

Rumus Kalimat Conditional Sentence Type 2 =

1. S + would + Verb 1 (infinitive) + If + S + Verb 2

Kalau kamu menaruh if clause di awal kalimat, maka gunakan tanda koma sebagai pemisah antara if clause dan main clause.

2. If + S + Verb 2, S + would + Verb 1 (infinitive)

Jadi secara garis besar pada kalimat conditional sentences type 2 kita menggunakan simple past untuk if clause dan simple past future pada main clause.

Catatan: untuk conditional sentences type 2, kita selalu menggunakan to be “were” meskipun subject atau pronoun yang digunakan tunggal (I, he, she, dan it).

1. If I were you, I would stay away from her.

Jika aku jadi kamu, aku akan menjauh darinya.

Fact: You keep close with her.

2. Indra would be very happy if his YouTube channel got one million subscribers.

Indra akan sangat senang jika kanal YouTube-nya mendapat satu juta pelanggan.

Fact: Indra’s YouTube channel doesn’t get one million subscribers so he isn’t very happy.

3.What would you do if you were stranded on a desert island?

Apa yang akan kau lakukan jika kau terdampar di pulau terpencil?

Fact: You are not stranded on a dessert island.

4. If Indah came 5 minutes earlier, she wouldn’t miss her bus.

Jika Indah datang 5 menit lebih awal, dia tidak akan ketinggalan bisnya.

Fact: Indah comes 5 minutes late so she misses her bus.

5. If I were a famous singer, I would perform a solo concert.

Jika aku seorang penyanyi terkenal, aku akan menyelenggarakan konser solo.

Fact: I’m not a famous singer so I don’t perform a solo concert.

30. Contoh soal dan jawaban serta penjelasan tentang conditional sentence type 3


If you had worked harder, you would have passed your exam.

had + V3 , would have V3

The type 3 conditional refers to an impossible condition in the past and its probable result in the past (type 3 digunakan u/ mengutarakan sesuatu yg tidak akan terjadi,biasanya ekspresi ketidakpuasan akan peristiwa yang sudah lewat)

31. Contoh soal kalimat conditional sentence type 2


If my father were rich, I would live in a big villa with swimming polo inside.

(Jika ayah saya kaya, saya akan tinggal di sebuah vila besar dengan kolam renang di dalamnya.)

Keterangan: Dalam kalimat pengandaian ini, if my father was rich merupakan dependent clause yang tidak akan memiliki makna jika berdiri sendiri, sedangkan i would live in a big villa with swimming polo inside merupakan independent clause yang tetap memiliki makna meskipun tidak diikuti dengan kalimat di depan.

36. apa yang membedakan antara conditional type 2 dan conditional sentence type 3 ???


Conditional type 2 menggunakan rumus

If + S + V2, S + would + V1

Contohnya:

If I came, she would be mad.

Conditional type 3 menggunakan rumus

If + S + had + V3, S + would have + V3

Contohnya:

If hadn't been sick, I wouldn't have missed the lesson.
conditional type 3 utk kejadian masa lampau atau simple past tense, kejadian yang harus sudah dipenuhi di masa lalu. Rumus Conditional Sentence Type 3
If +  Subjek + Had + V3 +S+ M.2 + Have + V.3

conditional type 2 ini digunakan utk kejadian di masa sekarang atau present yang akan terjadi jika kondisi yang ada berbeda.
Rumus Conditional Sentence Type 2
If + Subjek +V.2/Were +S + M2 + V.1M2 (Modal bentuk 2) : Would,could

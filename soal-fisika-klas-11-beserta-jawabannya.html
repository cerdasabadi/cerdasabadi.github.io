17. soal klas 7 yg bisa siapa ?​


Penjelasan dengan langkah-langkah:

[tex]3y + 102 = 180 \\ 3y = 180 - 102 \\ 3y = 78 \\ y = \frac{78}{3} = 26[/tex]

Nilai y adalah

Penyelesaian (By : NatasyaLisz)

Pertama - tama Mari kita lihat Gambar pada soal tersebut dan Mari ansumsikan

Sudut 3y = A

Sudut 102° = B

Mari kita lihat kembali pada soal, apa kalian sudah sadar? Ternyata !!! Sudut B dan Sudut A bersebrangan sehingga :

Sudut A + Sudut B = 180°

Sudut A = 180° - Sudut B

3y = 180° - 102°

3y = 78°

y = 26°

30. Fisika klas X.Pakai penjelasanya yah ,Gomawo​


DiketahuiMassa balok A ([tex]m_a[/tex]) = 1,2 kgMassa balok B ([tex]m_b[/tex]) = 1,8 kgPercepatan gravitasi (g) = 10 m/s²

DitanyaBesar percepatan system = ..?

Pembahasan

Dalam hal ini massa katrol diabaikan [tex](m_k = 0)[/tex]

Dalam gambar terlihat balok B menarik balok A dan sebaliknya. Karena massa Balok B lebih besar dari massa Balok A [tex](m_b > m_a)[/tex]. Sehingga akan diperoleh:

[tex]\sum F = \sum m ~.~a[/tex]

[tex]w_b - w_a = (m_a + m_b)~.~a[/tex]

[tex]a = \frac{w_b - w_a}{m_a + m_b}[/tex]

[tex]a = \frac{m_b.g ~-~ m_a.g}{m_a + m_b}[/tex]

[tex]a = g\times \frac{m_b - m_a}{m_a + m_b}[/tex]

Maka, Percepatannya adalah

[tex]a = g\times \frac{1,8~kg~-~ 1.2~kg}{1,8~kg ~+~ 1,2~kg}[/tex]

[tex]a = 10~m/s^2 \times \frac{0,6~kg}{3~kg}[/tex]

[tex]a = 10~m/s^2 \times 0,2[/tex]

[tex]a = \boxed{2 ~m/s^2}[/tex]

→ Jadi, percepatannya adalah 2 m/s²

38. please bantu ya Kaka ..fisika klas 7​


diket:

panjang awal (L0) = 200 cm

suhu awal = 20°C

suhu akhir = 220°C

koefisien muai panjang (a) = 0,000012/°C

ditanya:

panjang setelah pemanasan (L)...?

jawab:L = L0(1 + a ∆T)

L = 200 cm (1 + (0,000012/°C x (220°C - 20°C)

= 200 cm (1 + (0,000012/°C x 200°C)

= 200 cm (1 + 0,0024)

= 200 cm (1,0024)

= 200,48 cm

Jadi, panjang potongan besi setelah pemanasan adl 200,48 cm.

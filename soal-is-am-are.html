SIMPLE PRESENT TENSE

Simple : Sederhana

Present : Saat ini

Simple present tense adalah kalimat sederhana yang digunakan untuk menyatakan kejadian saat ini, seperti kegiatan sehari - hari atau kebenaran umum / fakta (general truth).

Dibagi menjadi 2 :

1. Verba : Menggunakan kata kerja.

2. Nominal : Menggunakan kata sifat (Adjective), kata benda (Noun), dan kata keterangan (Adverb).

RUMUS SIMPLE PRESENT TENSE

- Verbal ~

- Nominal, digunakan saat menggunakan kata sifat, kata benda, dan kata keterangan. Menggunakan to be (is, am, are)

Penggunaan to be :

Is : She, He, It

Am : I

Are : You, We, They.

• Possitive (+) : Subject + To be + Adj, Noun, Adv.

• Negative (-) : Subject + To be + Not + Adj, Noun, Adv.

• Interogative (?) : To be + Subject + Adj, Noun, Adv?

L = L tabung (tanpa tutup) + L kerucut (tanpa alas)

Diketahui

Soal Matematika

Perhatikan gambar di atas untuk menjawab pertanyaan berikut !

Soal pilihan ganda

1. Perhatikan bangun yang dibentuk oleh kerucut dan tabung seperti gambar diatas. Luas permukaan bangun tersebut adalah?

A. 572 cm²

B. 990 cm²

C. 1.064 cm²

D. 1.144 cm²

Jawab beserta langkah-langkahnya

Di Tanyakan

Luas permukaan bangun tersebut adalah?

Di jawab

D. 1.144 cm²

Pembahasan

Luas tabung tanpa tutup = luas alas + luas selimut tabung = π x r x r + 2π rt

Luas selimut kerucut = π r s

Langkah langkahnya yaitu kita cari t kerucut terlebih dahulu setelah hitung garis pelukis.

Baru kita cari alas tabung , selimut tabung, dan selimut kerucut.

Kesimpulan

Jadi, Luas permukaan bangun tersebut adalah D. 1.144 cm²

Diketahui

Soal pilihan ganda !

1. Perhatikan gambar di atas.

Bangun tersebut dibentuk oleh kerucut dan belahan bola. Volume bangun tersebut adalah?

A. 718,67 cm³

B. 1.232,67 cm³

C. 1.718,67 cm³

D. 1.950,67 cm³

jawab beserta langkah-langkahnya

Ditanyakan

Volume bangun tersebut adalah?

Di jawab

[tex]D. 1.950,67 cm³[/tex]

Pembahasan

Volume kerucut :

[tex]= \frac{1}{3} πr²t[/tex]

[tex]= \frac{1}{3} × \frac{22}{7} ×7×7×24[tex]

[tex]= 1232 \: cm³[/tex]

Volume setengah bola :

[tex]= \frac{1}{2} × \frac{4}{3} πr³[/tex]

[tex]= \frac{2}{3} × \frac{22}{7} ×7×7×7[/tex]

[tex]= \frac{2156}{3} [/tex]

[tex]= 718,67 \: cm³[/tex]

Volume gabungan :

= volume kerucut + volume setengah bola

[tex]= 1232 cm³ + 718,67 cm³[/tex]

[tex]= 1950,67 \: cm³ [/tex]

Jawaban:

Dik:

Jari - jari = 7cm

tinggi kerucut = 24cm

tinggi ½ bola = 7cm

Dit: volume gabungan?

Dijawab-

_Volume ½ bola_

=> ½ x 4/3 x π x r³

=> ½ x 4/3 x 22/7 x (7cm)³

=> ½ x 4/3 x 1.078

=> 718,67cm³

_Volume kerucut_

=> ⅓ x π x r² x t

=> ⅓ x 22/7 x (7cm)² x 24cm

=> ⅓ x 3.696cm³

=> 1.232cm³

_Volume gabungan_

=> 718,67cm³ + 1.232cm³

=> 1.950,67cm³

=550 + 154 + 440
